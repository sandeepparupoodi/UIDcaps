About AnimeSA

AnimeSA Encyclopedia is a simple platform created for anime lovers to explore, discover, and learn about various anime series from different genres.

This website allows users to browse top anime, explore genres, and add new anime entries to expand the collection.

This project is developed using HTML and CSS as part of a capstone project, focusing on clean UI design and user-friendly experience.
